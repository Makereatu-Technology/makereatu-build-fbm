First: Access the VM via RDP

Connect to the VM.

Open Remote Desktop (RDP) and log in to the VM.

Confirm the environment:

winver

You should see Windows Server 2022 (Desktop Experience).

2. Install Core Development Tools

These are required for the FBM pipeline (Python + OCR + extraction).

Install Python

Download Python:

https://www.python.org/downloads/windows/

During installation:

✔ Check Add Python to PATH

Verify installation:

python --version
pip --version
3. Create a Project Environment

Never install packages globally.

Create a virtual environment.

Navigate to your project folder:

cd C:\makereatu\fbm

Create environment:

python -m venv fbm_env

Activate it:

fbm_env\Scripts\activate

You should see:

(fbm_env)
4. Install Python Requirements

If your repo has a requirements.txt:

pip install -r requirements.txt

If not, install manually.

Typical FBM pipeline packages:

pip install pandas
pip install numpy
pip install scikit-learn
pip install pytesseract
pip install pillow
pip install pdf2image
pip install opencv-python
pip install psycopg2-binary
pip install sqlalchemy
pip install requests
pip install beautifulsoup4
pip install python-dotenv

For Azure:

pip install azure-ai-documentintelligence
pip install openai
pip install azure-identity

Verify:

pip list
5. Install Tesseract OCR

This is required for local OCR fallback.

Download Windows installer:

https://github.com/UB-Mannheim/tesseract/wiki

Install to:

C:\Program Files\Tesseract-OCR

Add to PATH if needed.

Test:

tesseract --version
6. Configure Tesseract for Python

In Python:

import pytesseract

pytesseract.pytesseract.tesseract_cmd = r"C:\Program Files\Tesseract-OCR\tesseract.exe"

Test OCR:

import pytesseract
from PIL import Image

text = pytesseract.image_to_string(Image.open("test.png"))
print(text)
7. Install PostgreSQL (Important for FBM pipeline)

Download:

https://www.postgresql.org/download/windows/

During installation:

User:

postgres

Set password.

Default port:

5432

Verify:

psql --version
8. Install Required CLI Tools

Barry mentioned these tools should already exist.

Verify:

Azure CLI
az version

If missing:

https://learn.microsoft.com/en-us/cli/azure/install-azure-cli

Power Platform CLI
pac --version
9. Clone Your FBM Pipeline Repo

Example:

git clone https://github.com/makereatu/fbm-pipeline.git

Then install requirements again:

pip install -r requirements.txt
10. Configure Environment Variables

Create .env

Example:

AZURE_OPENAI_ENDPOINT=
AZURE_OPENAI_KEY=
AZURE_DOC_INTELLIGENCE_ENDPOINT=
AZURE_DOC_INTELLIGENCE_KEY=
POSTGRES_HOST=
POSTGRES_DB=
POSTGRES_USER=
POSTGRES_PASSWORD=

Run a simple test:

python scripts/run_core_extraction.py

Makereatu FBM Core Extraction
